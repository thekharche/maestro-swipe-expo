## Run Maestro Tests

Find the tests in `.maestro` directory.

```
maestro test .maestro/flow.yaml
```

Install Maestro:

```
curl -Ls "https://get.maestro.mobile.dev" | bash
```

Maestro is a UI testing framework for Android | iOS.
